## Copyhawk component for your React applications

### Getting Started

  * Download the project
  * Start using the component in your app wherever you want Copyhawk content placements
  * Works great on both web apps and mobile apps
  * Current code works as-is and is connected to our Open Demo Site, use the component to test out our implementation

### Customization

This code is provided as a basic starting point to be optimized and improved as needed. Please let us know if any 
features you would like are missing and we will get to work.
